Obiettivo:
    Creare un programma in C che permetta di gestire le segnalazioni inviate dai
    cittadini a un comune, relative a problemi urbani come illuminazione guasta, buche
    stradali, rifiuti abbandonati o guasti agli impianti pubblici.
    Il sistema deve consentire di registrare, classificare, monitorare e aggiornare lo stato
    delle segnalazioni.

        // ○ data di inserimento
        // ○ livello di urgenza
        // ○ stato della segnalazione

   // ○ chiuse

    ○ numero totale di segnalazioni
